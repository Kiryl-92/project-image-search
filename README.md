# Поиск референсных фотографий по текстовому описанию

## Описание проекта
Фотохостинг «Со Смыслом» разрабатывает поиск референсных фотографий для фотографов. 
Пользователь вводит текстовое описание сцены, сервис выводит фотографии с похожей сценой.

## Цель
Разработать демонстрационную версию (PoC) поиска изображений по текстовому запросу. 
Модель должна получать векторное представление изображения и текста, на выходе выдавать 
число от 0 до 1, показывающее степень соответствия.

## Юридические ограничения
В некоторых странах запрещено обрабатывать изображения детей до 16 лет. В проекте реализована:
- Фильтрация данных от детского контента на этапе подготовки
- Проверка запросов пользователя и вывод дисклеймера при обнаружении запрещенного контента

## Реализованные шаги
1. ✅ Загрузка данных и исследовательский анализ данных (EDA)
2. ✅ Подготовка данных (фильтрация детского контента, векторизация)
3. ✅ Обучение и сравнение моделей
4. ✅ Тестирование и демонстрация работы

## Модели
- **Векторизация текста**: all-MiniLM-L6-v2 (SentenceTransformer) с mean-pooling, размерность 384
- **Векторизация изображений**: ResNet50 с весами ImageNet, размерность 2048
- **Нормализация**: L2-нормализация (Normalizer) отдельно для текстовых и визуальных эмбеддингов
- **Обученные модели**: 
  - Линейная регрессия (R² = -0.22 на валидации)
  - Нейронная сеть (PyTorch) с архитектурой 2432 → 256 → 128 → 64 → 1, Sigmoid + BCELoss
- **Лучшая модель**: Нейронная сеть (R² = 0.42, RMSE = 0.23, MAE = 0.17 на валидации)

## Результаты
- Проведена корректная фильтрация детского контента (сохранено 4,138 записей из 5,696)
- Достигнуто значительное улучшение качества модели (R² вырос с 0.15 до 0.42)
- Реализована функция поиска изображений с проверкой на запрещенный контент
