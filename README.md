# Поиск референсных фотографий по текстовому описанию

## Описание проекта
Фотохостинг «Со Смыслом» разрабатывает поиск референсных фотографий для фотографов. 
Пользователь вводит текстовое описание сцены, сервис выводит фотографии с похожей сценой.

## Цель
Разработать демонстрационную версию (PoC) поиска изображений по текстовому запросу. 
Модель должна получать векторное представление изображения и текста, на выходе выдавать 
число от 0 до 1, показывающее степень соответствия.

## Структура проекта
- `project_notebook.ipynb` - основной ноутбук с реализацией
- `train_dataset.csv` - обучающий датасет
- `train_images/` - папка с изображениями для обучения
- `test_queries.csv` - тестовые запросы
- `test_images/` - папка с изображениями для тестирования

## Реализованные шаги
1. ✅ Загрузка данных и исследовательский анализ данных (EDA)
2. ✅ Подготовка данных (фильтрация детского контента, векторизация)
3. ✅ Обучение и сравнение моделей
4. ✅ Тестирование и демонстрация работы

## Модели
- Векторизация текста: BERT (768 признаков)
- Векторизация изображений: ResNet50 (2048 признаков)
- Обученные модели: Линейная регрессия, Нейронная сеть (PyTorch)
